---

## ✨ Fitur Dashboard

- **Peta Sebaran** — Visualisasi interaktif persebaran alumni di seluruh Indonesia & luar negeri
- **Database Alumni** — Tabel lengkap dengan filter angkatan & status, pencarian nama
- **Rincian Alumni** — Profil detail per alumni: pendidikan, karier, pengalaman akademik, organisasi
- **Login Verifikasi** — Akses terproteksi dengan verifikasi nama + angkatan + tanggal lahir
- **Live Data** — Terhubung langsung ke Google Sheets, data selalu up-to-date
- **4 Tema Warna** — Dark green, light green, light blue, blue dark

## ✨ Fitur Form

- **Registrasi Baru** — Pengisian data lengkap dalam 8 langkah
- **Update Data** — Alumni yang sudah terdaftar bisa memperbarui informasi
- **Verifikasi Duplikat** — Sistem OTP via email untuk cegah data ganda
- **Multi-entry** — Mendukung lebih dari satu pekerjaan sampingan & organisasi

---

## 🛠️ Teknologi

- HTML, CSS, JavaScript murni — tanpa framework
- Google Sheets sebagai database (published as CSV)
- Google Apps Script untuk backend form submission & OTP
- Hosted via GitHub Pages
